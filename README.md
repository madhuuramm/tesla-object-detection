# tesla-object-detection
# Tesla Autopilot Clone – Object Detection

## Overview
This project implements a real-time object detection system similar to Tesla Autopilot using YOLOv8.

## Features
- Real-time object detection
- Detects vehicles, pedestrians, traffic objects
- Works with webcam or video input

## Installation
pip install -r requirements.txt

## Run
python detect.py

## Model Used
YOLOv8 pretrained model

## Results
- Successfully detects objects in real time
- High accuracy with fast performance

## Challenges
- Lighting variations
- Small object detection

## Improvements
- Use custom dataset
- Add lane detection
- Integrate GPS
## GUI Application

This project includes a Tkinter-based GUI that allows users to:

- Start real-time object detection via webcam
- Upload and analyze video files
- Stop detection anytime

The GUI makes the system user-friendly and interactive.
